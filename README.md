# 🌸 Flora - Flowers Website

A beautiful, responsive flowers website built as a markup practice project. This project showcases modern web development techniques using HTML, CSS, and JavaScript, with animations and interactive elements.

## ✨ Features

- **Hero Section**: Eye-catching introduction with stunning visuals
- **About Us**: Information about the flower shop and its story
- **Bestsellers**: Highlighting popular flower arrangements
- **Bouquets**: Extensive collection of flower bouquets with images and details
- **Customer Feedback**: Testimonials and reviews from satisfied customers
- **Contact Information**: Easy-to-find contact details and location
- **Interactive Modals**: Product details and order forms in modal windows
- **Responsive Design**: Optimized for all device sizes
- **Smooth Animations**: Powered by AOS (Animate On Scroll) library

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with CSS variables and modular architecture
- **JavaScript**: Interactive functionality and DOM manipulation
- **Vite**: Fast development server and build tool
- **AOS Library**: Scroll animations
- **Modern Normalize**: Consistent cross-browser styling

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/UMT-markup-practice-Dymchevskyi.git
   cd UMT-markup-practice-Dymchevskyi
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` (or the port shown in the terminal).

### Build for Production

To build the project for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

After building, you can preview the production build locally:

```bash
npm run preview
```
